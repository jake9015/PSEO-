# SOZEE LANDING PAGE FORMULAS & VARIABLES

## LANDING PAGE PATTERNS (8 Total)

### Pattern 1A: Competitor Comparison by Audience
**Formula:** `/sozee-vs-{competitor}-for-{audience}`
**H1:** `Sozee vs {Competitor} for {Audience}`
**Variables:** competitor, audience
**Example:** /sozee-vs-higgsfield-for-onlyfans-creators

### Pattern 1B: Competitor Comparison by Platform
**Formula:** `/sozee-vs-{competitor}-{platform}`
**H1:** `Sozee vs {Competitor} for {Platform}`
**Variables:** competitor, platform
**Example:** /sozee-vs-krea-onlyfans

### Pattern 2A: Best Tool for Audience
**Formula:** `/best-{use_case}-for-{audience}`
**H1:** `Best {Use_Case} for {Audience}`
**Variables:** use_case, audience
**Example:** /best-ai-photo-generator-for-onlyfans-agencies

### Pattern 8: Competitor Alternative
**Formula:** `/{competitor}-alternative-{audience}`
**H1:** `{Competitor} Alternative for {Audience}`
**Variables:** competitor, audience
**Example:** /higgsfield-alternative-onlyfans-creators

### Pattern 9: Product Review
**Formula:** `/sozee-review-{audience}`
**H1:** `Sozee Review for {Audience}`
**Variables:** audience
**Example:** /sozee-review-onlyfans-agencies

### Pattern 10: Direct Tool Search
**Formula:** `/ai-{tool_type}-{platform}`
**H1:** `AI {Tool_Type} for {Platform}`
**Variables:** tool_type, platform
**Example:** /ai-photo-generator-onlyfans

### Pattern 11: Platform-First Search
**Formula:** `/{platform}-{tool_type}`
**H1:** `{Platform} {Tool_Type}`
**Variables:** platform, tool_type
**Example:** /onlyfans-ai-photo-generator

### Pattern 12: Content Crisis Solution
**Formula:** `/content-crisis-solution-{audience}`
**H1:** `Solve the Content Crisis for {Audience}`
**Variables:** audience
**Example:** /content-crisis-solution-onlyfans-agencies
